C# & .NET Core

1. Inyeccion de dependencias

Instalar Entity Framework

  • Microsoft Entity Framework Core SqlServer
  • Microsoft Entity Framework Core Tools

2. Mapear la Base de Datos

Mapear la base de datos en la carpeta Models

  • Abrir Consola de Consola de Comandos (Herramientas>Administrador de Paquetes NuGet>Consola del Administrador de Paquetes)
  • Escribimos el siguiente comandos:

Comando:

PM> Scaffold-DbContext "Server="Server=(localdb)\MSSQLLocalDB; Database=BandList; Trusted_Connection=True;" Microsoft.EntityFrameworkCore.SqlServer -OutputDir Models"

3. Reubicar la cadena de conexion

Reubicar la cadena de conexion de Models.(tuContext)

  • 1. Ir a allsetting.json
  • 2. Agregar el nuevo campo "ConnectionStrings" y agregar la conexion: 

    {
        "Logging": {
            "LogLevel": {
            "Default": "Information",
            "Microsoft.AspNetCore": "Warning"
            }
        },
        "AllowedHosts": "*",
        "ConnectionStrings": {
            "BandListContext": "Server=(localdb)\\MSSQLLocalDB; Database=BandList; Trusted_Connection=True;"
        }
    }
  • Agregamos el DbContext en el Program.cs.
    
        builder.Services.AddDbContext<BandListContext>(options =>
        {
            options.UseSqlServer(builder.Configuration.GetConnectionString("BandListContext"));
        });

Agregamos un controlador

  • Abrir el Explorador de Soluciones
  • Click Derecho sobre la carpeta Controllers
  • Agregar>Controlador...
  • Seleccionamos la opcion Controlador de MVC: en blanco y 'Agregar'
  • Le cambiamos el nombre: nombre_del_controlador+Controller.cs, por ejemplo ArtistController.cs

- @Html.Action()

@Html.Action(String actionName, String controlName), Genera una dirección URL completa para un método de acción mediante el metodo de acción y el nombre de la ruta especificado.

Parametros:

- actionName : String

El nombre del método de acción.

- controlName : String

El nombre del controlador.

Retorna:

String

Dirección URL completa para un método de acción.

- jQuery.getJSON()

jQuery.getJSON( url [, data ] [, success ] ), Cargar datos codificados en JSON desde el servidor mediante una petición HTTP GET.

Parametros:

- url : String

Cadena que contiene la URL a la que se envía la solicitud.

- data : String

Un objeto plano o cadena que se envía al servidor con la solicitud.

- success : function

Una función de devolución de llamada que se ejecuta si la solicitud tiene éxito.

- .change()

.change( handler ), Vincula un manejador de eventos al evento "cambio", o dispara ese evento en un elemento..

Parametros:

- handler : function

Una función que se ejecutará cada vez que se active el evento.

.change( eventData, handler )

Parametros:

- eventData : Anything

Un objeto que contiene los datos que se pasarán al manejador del evento.

- handler : function

Una función que se ejecutará cada vez que se active el evento.
